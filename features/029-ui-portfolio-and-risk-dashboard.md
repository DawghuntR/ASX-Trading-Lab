# 029 - UI: Portfolio & Risk Dashboard

Provide UI pages showing paper portfolio performance and risk status.

## Description

- Display:
  - current positions and exposure
  - equity curve and daily P/L
  - drawdown and streak metrics
  - risk rule status (OK / breached)

## Impact

- Makes the “discipline coach” visible and usable.

## Success Criteria

- Portfolio and risk data load via Supabase read-only access.
- Key metrics render correctly.
- Risk breaches are clearly indicated.
