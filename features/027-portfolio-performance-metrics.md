# 027 - Portfolio Performance Metrics

Compute portfolio-level performance metrics from the paper trading ledger.

## Description

- Calculate daily P/L and equity curve.
- Compute basic metrics:
  - total return
  - max drawdown
  - win rate (trade-level)
  - exposure over time

## Impact

- Turns a ledger into actionable evaluation.

## Success Criteria

- Metrics can be computed from stored paper trades.
- Metrics are persisted or viewable via queries.
- UI can display at least an equity curve and drawdown.
